# LexiScan - AI Legal Document Analysis

LexiScan is a clean, modern, and professional legal-tech SaaS web application designed to simplify legal documents instantly. Upload contracts to classify clauses, detect risks, and get AI-powered explanations of complex legal language.

## Features

- **Clause Classification**: Automatically identifies payment, termination, liability, and other key clauses.
- **Risk Detection**: Highlights potentially risky terms using explainable logic.
- **AI Explanation**: Provides simplified, plain-language explanations of legal jargon.
- **Interactive Assistant**: A dedicated legal assistant chat to answer questions about your documents.
- **Modern UI**: Built with a focus on readability and professional enterprise aesthetics.

## Tech Stack

- **Frontend**: React (Vite)
- **Styling**: Tailwind CSS v4
- **UI Components**: shadcn/ui (Radix UI)
- **Icons**: Lucide React
- **Routing**: wouter
- **State Management**: TanStack Query (React Query)
- **Animation**: Framer Motion

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- npm (pkg manager)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/lexiscan-ai.git
   cd lexiscan-ai
2. Install dependencies:
   ```bash
   cd frontend
   npm install
   ```

### Development

Run the development server:
```bash
npm run dev
```
The application will be available at [http://localhost:5173/](http://localhost:5173/).
